      1. I am a registered user, but unable to execute a query, why?

        2. This is because the basic user does not have algorithm execution permissions. To run a query, the user must request an upgrade to the IRB verified (or super user). Please visit FAQ #2 to learn more about User Types and their privileges.

      2. How many User Types are there and what are their privileges?

        3. We have three types of users with defined roles and responsibilities. The types of users and their privileges are as follows:

        • Basic User: The basic user has read-only permissions for viewing algorithms (phenotype criteria) and the ability download algorithms.
        • IRB Verified User: The type of user is known as super-user and must have completed IRB training. The web-administrator will verify the IRB number while upgrading the user privileges from basic to super user. These types of users can upload new algorithms, execute and view query results, as well as download reports and data source.
        • Admin User: This user administers the whole website and reviews all operations of phenotype portal. They have full access to site management functions such as change/update phenotype tree, delete existing algorithms, re-assign/change phenotype tree classes for an existing algorithm.

      3. What kind of files does phenotype portal accept when uploading an algorithm?

        4. This portal only accepts a zip file which contains xml, xsd, and html file types generated by NQFâ€™s MAT tool.

      4. What types of files will be downloaded if I download a phenotype?

        • You will receive a zip file which contains xml, xsd and html files if you download the phenotype from the Criteria Tab
        • You will receive a pdf and excel file if you are downloading the phenotype from the Summary, Demographics or Flowchart Tabs.

      5. Do you have a common log-in without registering the real name and email similar to the I2b2.org link http://webservices.i2b2.org/webclient/?

        6. No. We want every user to register to enjoy the functionality of this application.

      6. Can I run this app in my handheld device?

        7. Not at this time.

      7. Can I print demographics and summary charts and flow charts?

        8. Yes. Please click on the chart tabs (i.e Summary, Demographics, Flowchart), find "Print" button located in bottom-right of the screen and click it.

      8. Are there any restrictions selecting the date range for algorithm execution? For example, can I execute an algorithm against a database for a date range of five years or more?

        9. There are not any date range restrictions, but please note that a large date range will result a longer query/wait time.. The application will email you if the query takes more than a minute and inform you that the query is running in the background.
