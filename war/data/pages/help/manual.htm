2 - Phenotype Selection and Query

Go to Top

2.1 - Select Phenotype Page

Go to Top

1. Select the "Phenotypes" link (top middle of the screen)

2. The following page will show up where the left side will show the group of disease categories.

3. Choose the disease category and double click on it.

4. The selected category will expand and show all its subcategories.

5. Keep double clicking until you find your phenotype.

6. Once you find it, double click on it, the criteria for the algorithm will display in right pane.

7. This page contains the following tabs: File Info, Criteria, Summary, Demographics and a Workflow Chart.

8. However, at this time only, "Criteria" and "File info" tabs will be visible. The remaining tabs (Summary, Demographic and Workflow) will be enabled only after execution of query.

9. Note that the "Criteria" tab displays the detailed information about the disease's algorithm criteria.

10. Each phenotype criteria will have it own unique "AND"/"OR" decision logic which will be executed against the database to return the data for your query.

11. And the "File Info" tab displays the information about your phenotypes such as File name, Version, Description etc.


2.2 - Select Date & Query

Go to Top

This section will show you how to select a date range of data to execute your query against. Please keep in mind, only users with IRB authentication are allowed to run queries. The steps are as follows.

1. Select your algorithm following the steps described in the "Select Phenotype Page" section.

2. Go to the Date Range panel above the criteria tab.

3. Select your date range, For example June 1, 1995 ti June 1, 2012. This is the date range which the application will query in database.

4. Make sure you are choosing correct the dates (from/to), as this is one of the key inputs for this application.

5. Click the "Execute" button

6. After a successful execution, the tool will display following message:

7. You will now see the Summary, Demographics and Workflow tabs are enabled.

8. To view reports in graphical reports on your data, please click on these tabs:


3 - Display Results

Go to Top

Once you successfully run the query, the results will be displayed in as Summary and Demographics and flow chart tabs. Every tab has different graphs and representation of data. Details are as follows:

3.1 - Summary Tab

Go to Top

This tab will display the summary level patient data for each algorithm criteria, For example: Initial Patient Population, Denominator, Numerator, Exclusion, Exceptions, etc.


3.2 - Demographics Tab

Go to Top

This tab provides more detailed statistics regarding each category displayed in the "Summary Tab". It will display demographic level data such as: Gender, Age, Ethnicity and Race.

You can use the pull down menu to select a demographic type.

For example, if you select Denominator you will see following graphs:

This tab displays the summary level data for the patients of each data set. For example, Denominator Criteria has unique demographic categories such as gender, age, ethnicity, and race. In this example, gender has two permissible values; male and female.
