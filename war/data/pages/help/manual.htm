2 - Phenotype Selection and Query

Go to Top

2.1 - Select Phenotype Page

Go to Top

1. Select the "Phenotypes" link (top middle of the screen)

2. The following page will show up where the left side will show the group of disease categories.

3. Choose the disease category and double click on it.

4. The selected category will expand and show all its subcategories.

5. Keep double clicking until you find your phenotype.

6. Once you find your phenotype, double click on it, the criteria for the algorithm will display in right pane.

7. This page contains the following tabs: File Info, Criteria, Summary and Demographics.

8. However, at this time only, "Criteria" and "File info" tabs will be visible. The remaining tabs (Summary & Demographic) will be enabled only after execution of query.

9. The "File Info" tab displays the information about your phenotypes such as File name, Version, Description, etc.

10. The "Criteria" tab displays the detailed information about the disease's algorithm criteria.

12. Each phenotype criteria will have it own unique "AND"/"OR" decision logic which will be executed against the database to return the data for your query.

2.2 - Edit Value Sets Page

Go to Top

This section will show you how to edit a phenotype's value sets.

1. Click the value set edit button in the value sets grid.

2. The Value Set Editor window will be displayed.

2.1 Click the arrow in the upper left to show the codes in the value set.

2.2 Click the edit button to edit the codes.

3. The edit codes window will be displayed.

3.1 To search for codes to add to the selected value set.

1. To search for codes to add first select the Code System to search within. (#1)

2. Select the Code System Version to search within. (#2)

3. Enter the search term and press 'Enter'. (#3)

4. To add codes to the value set check the check box on the right of each desired row.

5. The selected codes will be added to the Selected Entities grid.

3.2 To remove codes from the selected value set either uncheck the checkbox in the search results grid or click the delete icon in the selected entities grid.

3.3 Click the finish button when you are done editing the value set.

3.4 To save the edited value set as a new version click the SaveAs... button.

3.5 Enter a comment in the Save As window and click the Save button.

4. To select which version of the value set to use click the value set version button.

4.1 Select the version you wish to use from the value set version window.
